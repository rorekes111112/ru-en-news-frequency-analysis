# ru-en-news-frequency-analysis
Этот проект представляет собой инструмент для автоматизированного частотного анализа слов в новостном дискурсе на русском и английском языках. Анализ выполняется на корпусе текстов новостных статей и включает лемматизацию, фильтрацию стоп-слов, нормализацию лексики и расчёт частотных показателей.
