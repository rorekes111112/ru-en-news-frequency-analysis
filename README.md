# ru-en-news-frequency-analysis
Этот проект представляет собой инструмент для автоматизированного частотного анализа слов в новостном дискурсе на русском и английском языках. Анализ выполняется на корпусе текстов новостных статей и включает лемматизацию, фильтрацию стоп-слов, нормализацию лексики и расчёт частотных показателей.  
Проект используется в рамках курсовой работы по корпусной лингвистике и дискурс-анализу.

Основные возможности:
обработка русскоязычного и англоязычного текстового корпуса
лемматизация:
   русский: pymorphy3
   английский: spaCy
удаление стоп-слов (NLTK + расширенные списки)
ручная лексическая нормализация (например, russian → russia)
расчёт:
   абсолютной частоты
   относительной частоты (на 1000 слов)
   статистики корпуса
   экспорт результатов в Excel

Структура корпуса:
Ожидается следующая структура директорий:
project/
│
├── ru_articles/
│   ├── article1.txt
│   ├── article2.txt
│   └── ...
│
├── en_articles/
│   ├── article1.txt
│   ├── article2.txt
│   └── ...

Результаты:
После запуска создаётся файл: corpus_analysis.xlsx
Он содержит 4 листа:
RU Top 100 — топ-100 лемм русского корпуса
EN Top 100 — топ-100 лемм английского корпуса
RU Statistics — общая статистика русского корпуса
EN Statistics — общая статистика английского корпуса

Используемые технологии:
Python 3.10+
pandas
nltk
spaCy (en_core_web_sm)
pymorphy3
collections.Counter

Установка зависимостей:
pip install pandas nltk spacy pymorphy3
python -m spacy download en_core_web_sm

Методология:
Токенизация текста
Лемматизация
Удаление стоп-слов
Ручная нормализация лексем
Подсчёт частот
Формирование топ-100 слов
Экспорт результатов

Примечание:
Корпус состоит из новостных текстов и используется для сравнительного анализа частотных паттернов в англоязычном и русскоязычном спортивном новостном дискурсе.
